Boas-vindas ao Proposal Flow!

Estes Termos de Uso ("Termos") regem o acesso e uso do Proposal Flow e fornecem informações sobre o serviço, observando exceções onde outros termos podem se aplicar.

O Proposal Flow é uma plataforma de automação de vendas que utiliza tecnologia OCR para processar documentos e automatizar processos. Ao acessar ou usar o Proposal Flow, você concorda com estes Termos.

1. O Serviço do Proposal Flow

O Proposal Flow concorda em fornecer seu serviço, que inclui todos os produtos, funcionalidades, aplicações, serviços, tecnologias e software, para fortalecer a automação de vendas e otimizar processos comerciais.

Oferecer oportunidades de automação de vendas.

O Proposal Flow oferece diferentes tipos de funcionalidades para ajudar corretores a automatizar processos de venda, processar documentos via OCR, extrair dados de clientes e executar automações. Utilizamos informações para destacar funcionalidades e otimizar a experiência de uso.

Promover um ambiente seguro e confiável.

Desenvolvemos e utilizamos ferramentas e recursos para tornar a experiência dos usuários segura, incluindo sistemas de autenticação, verificação por email, criptografia de dados e logs de auditoria.

2. Dados Coletados

Dados dos Corretores (Armazenados):

  • Nome, sobrenome, e-mail
  • Nome da corretora, telefone
  • Senha criptografada
  • Logs de acesso e ações

Dados dos Clientes (NÃO Armazenados):

  • CPF processado apenas durante automação
  • Dados pessoais extraídos do documento
  • Documentos processados e descartados imediatamente
  • Informações de venda usadas apenas para automação

Dados de Análise (Armazenados):

  • Métricas de venda por corretor
  • Tempo médio de processamento
  • Número de OCRs e automações realizadas
  • Dados agregados para análise de performance

3. Finalidade do Tratamento

Utilizamos os dados coletados para:

  • Automação de Processos: Processar documentos via OCR, extrair dados para automação, executar processos de venda e otimizar tempo de trabalho
  • Análise e Relatórios: Gerar estatísticas de vendas, analisar performance dos corretores, criar relatórios de produtividade e identificar tendências
  • Segurança e Auditoria: Controlar acesso ao sistema, registrar ações dos usuários, manter logs de auditoria e prevenir fraudes
  • Gestão de Usuários: Gerenciar contas de corretores, controlar permissões de acesso, manter dados de contato e facilitar comunicação

4. Compartilhamento de Dados

NÃO Compartilhamos Dados Pessoais

Os dados pessoais dos clientes NÃO são compartilhados, vendidos ou transferidos para terceiros. Eles são processados apenas para automação e descartados imediatamente após o uso.

5. Segurança dos Dados

Implementamos medidas de segurança para proteger seus dados:

  • Criptografia: Senhas criptografadas com bcrypt, conexões HTTPS/TLS, dados sensíveis protegidos e tokens JWT seguros
  • Banco de Dados: PostgreSQL seguro, backup automático, controle de acesso e logs de auditoria
  • Controle de Acesso: Autenticação obrigatória, controle de sessões, logout automático e permissões por função
  • Auditoria: Log de todas as ações, rastreamento de acessos, registro de alterações e monitoramento contínuo

6. Direitos do Titular (LGPD)

Conforme a Lei Geral de Proteção de Dados (LGPD), você tem os seguintes direitos:

  • Acesso: Solicitar informações sobre seus dados
  • Correção: Corrigir dados incorretos
  • Exclusão: Solicitar exclusão de dados
  • Portabilidade: Transferir dados para outro sistema
  • Anonimização: Tornar dados anônimos
  • Revogação: Revogar consentimento

7. Retenção de Dados

  • Dados dos Corretores: Mantidos enquanto a conta estiver ativa + 5 anos para fins de auditoria
  • Dados dos Clientes: NÃO são armazenados - processados e descartados imediatamente
  • Logs de Auditoria: Mantidos por 5 anos para conformidade legal
